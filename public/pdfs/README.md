# PDFs & Resources

Add PDF files to this folder and they will automatically appear in the Resources page (/pdfs) with an embedded viewer.

## How to use

1. Add your PDF files to this directory (`public/pdfs/`)
2. The PDFs will automatically appear in the Resources section of the app
3. Users can view PDFs inline and download them

## File naming

- Use descriptive names for your PDFs
- Hyphens and underscores in filenames will be converted to spaces in the UI
- Example: `nursing-study-guide.pdf` → "nursing study guide"

## Example files

Place your PDFs here:
- study-materials.pdf
- clinical-guidelines.pdf
- reference-notes.pdf
